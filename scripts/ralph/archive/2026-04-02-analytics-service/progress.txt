- OrthoEvent is in packages/@ortho/event-bus/src/types.ts — add new fields there first, then use them in analytics handlers
- All services use ESM ("type": "module") with NodeNext module resolution
- Handlers signature: (event: OrthoEvent, pool: Pool) => Promise<void>

- Use knex.raw() for PARTITION BY RANGE DDL and default partition creation; drop default partition before parent table in down()
